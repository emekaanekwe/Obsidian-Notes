## Exercise 5A: Substitution into Formulas and Tables of Values

### Building Understanding

**1. Removalist cost: \(C = 300 + 120t\)**

a) \(t = 1\): \(C = 300 + 120(1) = 420\) dollars  
b) \(t = 2\): \(C = 300 + 120(2) = 540\) dollars

**Tutoring Tips:**  
- Emphasize "substitute" means replace the variable with the given number.  
- Show order of operations: multiplication before addition.

---

**2. Perimeter of rectangle: \(P = 2L + 2W\)**

a) \(L = 4, W = 2\): \(P = 2(4) + 2(2) = 8 + 4 = 12\) cm  
b) \(L = 5.8, W = 3.5\): \(P = 2(5.8) + 2(3.5) = 11.6 + 7 = 18.6\) cm  
c) For length 7.9 cm and width 2.7 cm: substitute \(L = 7.9, W = 2.7\)

**Tutoring Tips:**  
- Remind students that perimeter is the distance around the outside.  
- Units (cm) must be included in the final answer.

---

**3. Cost of pies: \(C = 3x\)**

a) 43 pies: \(C = 3 \times 43 = 129\) dollars  
b) 44 pies: \(C = 3 \times 44 = 132\) dollars  

Table:

| x | 40 | 41 | 42 | 43 | 44 | 45 |
|---|---|---|---|---|---|---|
| C($) | 120 | 123 | 126 | 129 | 132 | 135 |

**Tutoring Tips:**  
- Show the pattern: adding 1 pie increases cost by $3.  
- Use this to check arithmetic: 120 → 123 → 126, etc.

---

### Developing Understanding

**4. Dance hall: \(C = 1200 + 50t\)**

a) \(t = 4\): \(C = 1200 + 200 = 1400\) dollars  
b) \(t = 6\): \(C = 1200 + 300 = 1500\) dollars  
c) \(t = 4.5\): \(C = 1200 + 225 = 1425\) dollars

**Tutoring Tips:**  
- Point out that 4.5 hours is allowed—formulas work with decimals.  
- Multiply carefully: \(50 \times 4.5 = 225\).

---

**5. Distance: \(d = v \times t\)**  
\(v = 95\) km/h, \(t = 4\) h: \(d = 95 \times 4 = 380\) km

**Tutoring Tips:**  
- Reinforce that speed × time = distance.  
- Check units: km/h × h = km.

---

**6. Taxi fare: \(F = 4 + 1.3K\)**

a) \(K = 5\): \(F = 4 + 1.3 \times 5 = 4 + 6.5 = 10.50\) dollars  
b) \(K = 8\): \(F = 4 + 1.3 \times 8 = 4 + 10.4 = 14.40\) dollars  
c) \(K = 20\): \(F = 4 + 1.3 \times 20 = 4 + 26 = 30.00\) dollars

**Tutoring Tips:**  
- Explain that the $4 is the flagfall (fixed cost).  
- Multiply decimals carefully: \(1.3 \times 20 = 26\).

---

**7. Circumference: \(C = 2\pi r\)** (use \(\pi \approx 3.1416\))

a) \(r = 3\) mm: \(C = 2 \times 3.1416 \times 3 = 18.85\) mm (2 d.p.)  
b) \(r = 7.2\) m: \(C = 2 \times 3.1416 \times 7.2 = 45.24\) m (2 d.p.)

**Tutoring Tips:**  
- Use calculator’s \(\pi\) button for accuracy.  
- Round only at the final step.

---

**8. Area of rectangle: \(A = L \times W\)**

a) \(L = 3, W = 4\): \(A = 12\) square units  
b) \(L = 15, W = 8\): \(A = 120\) square units

**Tutoring Tips:**  
- Area units are squared (cm², m², etc.).  
- Simple multiplication—good for building confidence.

---

**9. Area of trapezium: \(A = \frac{1}{2} h (a + b)\)**

a) \(h = 1, a = 3, b = 5\): \(A = \frac{1}{2} \times 1 \times (3 + 5) = \frac{1}{2} \times 8 = 4\)  
b) \(h = 5, a = 2.5, b = 3.2\): \(A = \frac{1}{2} \times 5 \times (2.5 + 3.2) = \frac{1}{2} \times 5 \times 5.7 = \frac{1}{2} \times 28.5 = 14.25\)

**Tutoring Tips:**  
- Add \(a + b\) first (brackets first).  
- Multiply by \(h\), then halve (or halve then multiply).

---

**10. Fahrenheit to Celsius: \(C = \frac{5}{9}(F - 32)\)**

a) \(F = 50\): \(C = \frac{5}{9}(18) = 10.0^\circ\)C  
b) \(F = 0\): \(C = \frac{5}{9}(-32) = -17.8^\circ\)C  
c) \(F = 212\): \(C = \frac{5}{9}(180) = 100.0^\circ\)C  
d) \(F = 92\): \(C = \frac{5}{9}(60) = 33.3^\circ\)C

**Tutoring Tips:**  
- Subtract 32 first.  
- Multiply by 5, then divide by 9 (or use fraction on calculator).

---

**11. Simple interest: \(I = \frac{PRT}{100}\)**

a) \(P = 5000, R = 12, T = 4\): \(I = \frac{5000 \times 12 \times 4}{100} = \frac{240000}{100} = 2400\) dollars  
b) \(P = 8500, R = 7.9, T = 3\): \(I = \frac{8500 \times 7.9 \times 3}{100} = \frac{201450}{100} = 2014.50\) dollars

**Tutoring Tips:**  
- Multiply all three numbers first, then divide by 100.  
- Remind them to round to two decimal places (nearest cent).

---

**12. Australian football points: \(P = 6g + b\)**

a) i) 2 goals, 3 behinds: \(P = 6 \times 2 + 3 = 12 + 3 = 15\) points  
   ii) 8 goals, 20 behinds: \(P = 48 + 20 = 68\) points  
b) Redteam: 4 goals, 2 behinds → \(24 + 2 = 26\) points  
   Greenteam: 3 goals, 10 behinds → \(18 + 10 = 28\) points  
   **Greenteam wins.**

**Tutoring Tips:**  
- Clarify that goals are worth more—use the formula to compare fairly.

**14. Phone bill: \(C = 40 + 0.18n\)**

### Testing Understanding

**18. Matchstick pattern: \(M = a + (n - 1)d\), \(a = 3, d = 2\)**

a) \(n = 6\): \(M = 3 + (5) \times 2 = 3 + 10 = 13\) matches  
b) \(n = 11\): \(M = 3 + 10 \times 2 = 3 + 20 = 23\) matches  
c) \(n = 50\): \(M = 3 + 49 \times 2 = 3 + 98 = 101\) matches

**Tutoring Tips:**  
- This is an arithmetic sequence formula.  
- Emphasize: \((n - 1)\) is the number of steps from the first term.

**19. Roasting times**

a) i) Chicken (45 min/kg + 20 min): 2 kg → \(45 \times 2 + 20 = 90 + 20 = 110\) minutes  
   ii) Beef well done (65 min/kg + 30 min): 2.25 kg → \(65 \times 2.25 + 30 = 146.25 + 30 = 176.25 \approx 176\) minutes  
   iii) Lamb well done (65 min/kg + 30 min): 2.4 kg → \(65 \times 2.4 + 30 = 156 + 30 = 186\) minutes  
   iv) Beef medium (55 min/kg + 20 min): 2.5 kg → \(55 \times 2.5 + 20 = 137.5 + 20 = 157.5 \approx 158\) minutes

b) 2 kg leg of lamb (medium: 55 min/kg + 25 min): \(55 \times 2 + 25 = 110 + 25 = 135\) minutes (2 hours 15 minutes)  
   If serving at 6:00 PM, put in oven at 3:45 PM.

**Tutoring Tips:**  
- Break into steps: multiply kg by minutes/kg, then add the fixed time.  
- For cooking start time, subtract total minutes from serving time.  
- Remind students to round to nearest minute as instructed.

## Exercise 5B: Solving Linear Equations and Developing Formulas

### Building Understanding

**1. Simple one-step equations (addition/subtraction)**

| a) \(x + 6 = 15\) | \(x = 9\) |
| b) \(y + 11 = 26\) | \(y = 15\) |
| c) \(m - 5 = 1\) | \(m = 6\) |
| d) \(m - 5 = -7\) | \(m = -2\) |
| e) \(6 + e = 9\) | \(e = 3\) |
| f) \(-n + 5 = 1\) | \(-n = -4 \Rightarrow n = 4\) |

**Tutoring Tips:**  
- Inverse operations: add → subtract, subtract → add.  
- For f), remind them to move the variable first or be careful with negatives.

---

**2. Simple one-step equations (multiplication/division)**

| a) \(5x = 15\) | \(x = 3\) |
| b) \(3g = 27\) | \(g = 9\) |
| c) \(6j = -24\) | \(j = -4\) |
| d) \(\frac{r}{3} = 4\) | \(r = 12\) |
| e) \(\frac{t}{-2} = 6\) | \(t = -12\) |
| f) \(\frac{h}{-8} = -5\) | \(h = 40\) |

**Tutoring Tips:**  
- Multiply to undo division, divide to undo multiplication.  
- For e) and f), emphasize that negative signs "cancel" when dividing negatives.

---

**3. \(2a + 15 = 27\)**

a) First step: Subtract 15 from both sides.  
b) Second step: Divide both sides by 2.  
c) Solution: \(2a = 12 \Rightarrow a = 6\)

**Tutoring Tips:**  
- Teach "reverse order of operations": undo addition/subtraction first, then multiplication/division.

---

**4. \(\frac{y}{4} - 10 = 0\)**

a) First step: Add 10 to both sides.  
b) Second step: Multiply both sides by 4.  
c) Solution: \(\frac{y}{4} = 10 \Rightarrow y = 40\)

**Tutoring Tips:**  
- Isolate the fraction term first.

---

### Developing Understanding

**5. Mixed one-step equations**

| a) \(\nu + 7 = 2\) | \(\nu = -5\) |
| b) \(9 - k = 2\) | \(k = 7\) |
| c) \(3 - a = -5\) | \(a = 8\) |
| d) \(-5b = -25\) | \(b = 5\) |
| e) \(13 = 3r - 11\) | \(24 = 3r \Rightarrow r = 8\) |
| f) \(\frac{x + 1}{3} = 2\) | \(x + 1 = 6 \Rightarrow x = 5\) |

**Tutoring Tips:**  
- For b) and c), rewrite as \(9 - k = 2 \Rightarrow 9 - 2 = k\) to avoid sign errors.  
- For e), variable on right is fine—just solve normally.

---

**6. Equations with brackets**

| a) \(2(y - 1) = 6\) | \(y - 1 = 3 \Rightarrow y = 4\) |
| b) \(8(x - 4) = 56\) | \(x - 4 = 7 \Rightarrow x = 11\) |
| c) \(3(g + 2) = 12\) | \(g + 2 = 4 \Rightarrow g = 2\) |
| d) \(3(4x - 5) = 21\) | \(4x - 5 = 7 \Rightarrow 4x = 12 \Rightarrow x = 3\) |
| e) \(8(2x + 1) = 16\) | \(2x + 1 = 2 \Rightarrow 2x = 1 \Rightarrow x = 0.5\) |
| f) \(3(5m - 2) = 12\) | \(5m - 2 = 4 \Rightarrow 5m = 6 \Rightarrow m = 1.2\) |
| g) \(\frac{2(a - 3)}{5} = 6\) | \(2(a - 3) = 30 \Rightarrow a - 3 = 15 \Rightarrow a = 18\) |
| h) \(\frac{4(r + 2)}{6} = 10\) | \(4(r + 2) = 60 \Rightarrow r + 2 = 15 \Rightarrow r = 13\) |

**Tutoring Tips:**  
- Encourage dividing both sides by the outside number first (rather than expanding) when possible.  
- For fractions, multiply both sides by the denominator first.

**7. Variables on both sides**

| a) \(2x = x + 5\) | \(x = 5\) |
| b) \(2a + 1 = a + 4\) | \(a = 3\) |
| c) \(4b - 10 = 2b + 8\) | \(2b = 18 \Rightarrow b = 9\) |
| d) \(7 - 5y = 3y - 17\) | \(7 + 17 = 3y + 5y \Rightarrow 24 = 8y \Rightarrow y = 3\) |
| e) \(3(x + 5) - 4 = x + 11\) | \(3x + 15 - 4 = x + 11 \Rightarrow 3x + 11 = x + 11 \Rightarrow 2x = 0 \Rightarrow x = 0\) |
| f) \(6(c + 2) = 2(c - 2)\) | \(6c + 12 = 2c - 4 \Rightarrow 4c = -16 \Rightarrow c = -4\) |
| g) \(2f + 3 = 2 - 3(f + 3)\) | \(2f + 3 = 2 - 3f - 9 \Rightarrow 2f + 3 = -3f - 7 \Rightarrow 5f = -10 \Rightarrow f = -2\) |
| h) \(5(1 - 3y) - 2(10 - y) = -10y\) | \(5 - 15y - 20 + 2y = -10y \Rightarrow -15 - 13y = -10y \Rightarrow -15 = 3y \Rightarrow y = -5\) |

**Tutoring Tips:**  
- Collect variable terms on one side, constants on the other.  
- Encourage checking the solution by substitution.

**8. CAS calculator equations (answers to 1 d.p.)**

| a) \(3a + 5 = 11\) | \(a = 2\) |
| b) \(4b + 3 = 27\) | \(b = 6\) |
| c) \(2w + 5 = 9\) | \(w = 2\) |
| d) \(7c - 2 = 12\) | \(c = 2\) |
| e) \(3y - 5 = 16\) | \(y = 7\) |
| f) \(4f - 1 = 7\) | \(f = 2\) |
| g) \(3 + 2h = 13\) | \(h = 5\) |
| h) \(2 + 3k = 6\) | \(k = 1.3\) |
| i) \(-4(g - 4) = -18\) | \(g - 4 = 4.5 \Rightarrow g = 8.5\) |
| j) \(\frac{2(s - 6)}{7} = 4\) | \(2(s - 6) = 28 \Rightarrow s - 6 = 14 \Rightarrow s = 20\) |
| k) \(\frac{5(t + 1)}{2} = 8\) | \(5(t + 1) = 16 \Rightarrow t + 1 = 3.2 \Rightarrow t = 2.2\) |

**21. Amy and Ben meeting**

Distance apart: 17.2 km  
Combined speed: \(12 + 10 = 22\) km/h  
a) Time = \(\frac{17.2}{22} = 0.7818\) hours  
\(0.7818 \times 60 \approx 47\) minutes  
b) Ben: \(12 \times 0.7818 \approx 9.4\) km  
   Amy: \(10 \times 0.7818 \approx 7.8\) km  
Check: \(9.4 + 7.8 = 17.2\) km

**Tutoring Tips:**  
- Use \( \text{time} = \frac{\text{distance}}{\text{speed}} \) with combined speed.  
- Convert decimal hours to minutes by multiplying by 60.  
- Round distances to 1 decimal place as instructed.
