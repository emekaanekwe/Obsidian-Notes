# Terminology

  

## Perplexity

### Motivation

any good model assigns high probability to the natural text in the data. but as context grows, it gets exponentially smaller

	Using raw probabilities creates exponentially smaller units, so comparing sequences of different lengths is impossible

	Need a way to use N-th token analysis, i.e., the geometric mean of probability per token
### Informal

*measures how many equally likely choices he model thinks it has at each step*

**high** - model is choosing words more often from a uniform distribution
**low** - smaller distro. model is more confident

### Formal
*it is the inverse of the chain rule probability:*
$$P(x_1, ... ,x_N)=\prod^N_{n=1}P(x_n|x_1,...,x_n-1)^{-\frac{1}{N}}$$
